# Gull - 產品更動紀錄 (Changelog)

所有對 Gull 平台的技術與產品更動都會記錄於此。

---

## [1.2.5] - 2026-02-28
### Added
- **未讀通知系統 (Notification System)**:
    - 新增 `NotificationContext`，每 15 秒輪詢用戶相關訂單狀態變更。
    - 使用 `localStorage` 存儲訂單狀態快照與已關閉通知，實現跨頁面持久化。
    - 狀態變更觸發 Toast 通知，從頂部滑入，顯示商品名稱與狀態變化 (如「待接單 → 待付款」)。
    - Toast 包含「查看訂單」快捷連結，手動關閉後不再重複彈出。
    - Dashboard tab 在有未讀通知時顯示紅色圓點計數徽章（手機底部導覽與桌面側邊欄）。

## [1.2.0] - 2026-02-28
### Added
- **下架機制 (Delisting)**:
    - 新增 `DELISTED` 訂單狀態。
    - 許願者可主動下架 `OPEN` 狀態的願望，管理員可下架 `OPEN` / `MATCHED` 狀態。
    - 下架的願望不再出現在廣場，後端阻擋對已下架願望的接單。
    - 許願者可將已下架的願望重新上架為 `OPEN`。
    - 過期願望（`expected_shipping_date` 已過且 `auto_extend = false`）自動下架。
    - Dashboard 中，下架的願望歸類至「已完成」分頁。
    - Admin 後台可見下架按鈕，分類至已完成篩選。

## [1.1.5] - 2026-02-28
### Added
- **AI 購物推薦 (AI Insights)**: 
    - 首頁加入「幫我推薦最近夯什麼」動態卡片。
    - 結合魔法棒 (Wand2) 與星空漸變視覺，提升 AI 智慧感。
    - 實現 Mock AI 彈窗 (Modal)，展示日本 🇯🇵、美國 🇺🇸、泰國 🇹🇭 的當季夯貨。
    - 每項推薦品項皆能一鍵導航至市場尋找相關願望。

## [1.1.0] - 2026-02-28
### Added
- **品牌重塑 (Branding)**: 平台正式更名為 **Gull**，副標為「Gull 購快，世界沒界限」。
- **全新 Landing Page**: 
    - 實現了「分流引導」設計，區分許願者與旅遊者路徑。
    - 視覺全面升級，符合現代 Premium 網頁審美，加入動態模糊背景與進場動畫。
    - 在手機解析度下優化標題顯示。
- **時間維度管理 (Date Logic)**:
    - 許願流程新增「預期寄出日」與「自動延續一週」開關。
    - 廣場 (`/market`) 預設按寄出日期排序，最快到期的排前面。
    - 旅遊者可以在首頁輸入旅遊日期直接導航至帶有日期篩選的廣場。
- **訂單詳情優化**: 在代購接單前的「待接單」狀態，極簡化操作介面，僅保留「接下此單」按鈕。

### Changed
- **路由調整**: 將原本的 `/` 改為 `/market`，而 `/` 現在做為 Landing Page。
- **翻譯優化**: 修正了部分 P2P 相關過時文案，統一改為 Gull 品牌風格。

## [1.0.5] - 2026-02-27
### Added
- **平台服務費 (Platform Fee)**: 
    - 實現買家與代購兩端的費率計算。
    - 買家服務費門檻化（1000 TWD 作為分界點）。
- **採購證明要求 (Proof of Purchase)**: 
    - 許願者發布時可勾選：商品採購照（必填）、收據副本、型號比對。
    - 代購在 `ESCROWED` 狀態必須上傳照片後才能點擊「完成採買」。
- **匯率顯示**: 計算公式內所有金額均加上台幣 (NT$) 或原幣別單位，提高財務透明度。

### Fixed
- **翻譯修正**: 修正了 `ESCROWED` 狀態在中文翻譯中未對應的問題。
- **隱私控制**: 買家地址僅在付款 (`ESCROWED`) 後對接單代購揭露。
